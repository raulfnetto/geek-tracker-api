# Geek Tracker API 🚀

Esse é o backend do Geek Tracker, responsável por gerenciar o cadastro de usuários e obras geeks. Estou desenvolvendo com Python e integrado a um banco de dados PostgreSQL na nuvem (Neon).

## 🛠️ Tecnologias Utilizadas
* **Python** 
* **FastAPI:** Criação das rotas e do servidor
* **PostgreSQL (Neon):** Banco de dados em nuvem
* **psycopg2:** Comunicação entre o Python e o banco
* **Uvicorn:** Servidor local
